# Кодирование ID плательщика

Streamlit-приложение для анонимизации CSV-выгрузок: заменяет `ID плательщика`
на уникальный анонимный код (свой для каждого email) и удаляет столбец `Email`.

## Что делает приложение

1. Пользователь загружает CSV-файл.
2. Появляется кнопка **«Закодировать»**.
3. Приложение:
   - для каждого email считает детерминированный код вида `PID_<hash>`
     (одинаковый email → всегда одинаковый код, даже в разных файлах и запусках);
   - заменяет этим кодом значения в столбце `ID плательщика`;
   - удаляет столбец `Email` из файла.
4. Показываются первые 10 строк обработанной таблицы.
5. Появляется кнопка **«Скачать обработанный файл»**.

## Формат входного файла

- Разделитель столбцов: `;`
- Кодировка: `UTF-8` (с BOM)
- Обязательные столбцы: `Email`, `ID плательщика`

Если один из этих столбцов отсутствует, приложение покажет ошибку.

## Установка и запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

После запуска откроется браузер по адресу `http://localhost:8501`.

## Деплой на Streamlit Community Cloud

1. Залейте `app.py` и `requirements.txt` в репозиторий на GitHub.
2. Зайдите на [share.streamlit.io](https://share.streamlit.io) и войдите через GitHub.
3. Нажмите **New app**, выберите репозиторий и укажите `app.py` как главный файл.
4. Нажмите **Deploy** — приложение будет доступно по публичной ссылке.

## Примечания

- Код генерируется по формуле `PID_<sha256(email.lower().strip())[:12]>`,
  поэтому обратное восстановление email из кода невозможно, а сопоставление
  «email → код» остаётся стабильным между разными загрузками и запусками.
- Исходный email нигде не сохраняется и не попадает в итоговый файл.
